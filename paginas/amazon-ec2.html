Amazon EC2

O que é o Amazon EC2?

O amazon Elastic Compute Cloud (Amazon EC2) é um serviço da AWS que oferece servidores virtuais na nuvem com capacidade de computação ajustável conforme a necessida. Com o EC2, é possível criar e gerenciar instâncias (máquinas virtuais) de forma fácil, flexível e com total controle.

A principal vantagem do EC2 é a escabilidade, sendo capaz de aumentar ou reduzir os recursos conforme o uso, pagando apenas pelo que utilizar. Isso diminui os custos com infraestrutura física e acelera o desenvolvimento e a entrega de aplicações.

Como funciona o Amazon EC2?

Uma instáncia EC2 é um servidor virtual que roda na nuvem da AWS, podendo escolher entre vários tipos de instância, cada um com combinações específicas de CPU, memória, armazenamento e rede, ideais para diferentes necessidades de uso.

É possível ajustar a capacidade facilmente, aumentado ou reduzindo recursos conforme a demanda, como em períodos de pico ou tarefas que exigem muito processamento.

Principais recursos do Amazon EC2

Instâncias
Servidores virtuais escaláveis e configuráveis.
AMIs (Amazon Machine Images)
Modelos pré-configurados com sistema operacional e software.
Tipos de instância
Diversas combinações de CPU, memória e rede para diferentes necessidades.
Amazon EBS
Volumes de armazenamento persistente para dados.
Armazenamento de instâncias
Armazenamento temporário, ideal para dados transitórios.
Pares de chaves
Mecanismo seguro para autenticação em instâncias.
Grupos de segurança
Firewall virtual que controla o tráfego de entrada e saída.

Casos de uso

  • Hospedagem de aplicações web e APIs;
  • Ambientes de desenvolvimento e teste;
  • Análises e big data;
  • Processamento de imagens ou vídeos;
  • Execução de sistemas legados.

Benefícios do Amazon EC2

Escabilidade elástica
Lance centenas ou milhares de instâncias conforme necessário.
Custo sob demanda
Pague apenas pelo tempo de uso, sem necessidade de investimento inicial.
Flexibilidade de escolha
Tipos de instância otimizados para diferentes workloads.
Segurança integrada
Compatível com padrões como o PCI DSS, usado para dados de cartão de crédito.
Alta disponibilidade
Suporte a zonas de disponibilidade, balanceamento de carga e backups automáticos.
