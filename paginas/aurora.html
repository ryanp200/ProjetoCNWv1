Amazon Aurora

O que é

O Amazon aurora é ume mecanismo de banco de dados relacinal gerenciado que é compativel com Mysql e PostgreSQL sendo que as ferramentas e aplicações utilizadas no mysql e PostgreSQL podem ser usadas com o Aurora. O aurora oferecce até cinco vezes a do MySQL e até três vezes a taxa de processamento do PostgreSQL, sem exigir alterações na maioria das aplicações existentes.

Beneficios do microsoft aurora

Alta performance

A aurora possui aplicações de alta persormace e worloads criticas possuindo total compatibolidade com o mysql e o PostgreSQL

Alta disponibilidade

É possivel construir as aplicações com Mult-AZ apoiado por um SLA de 99.99% de tempo de atividade e replicação global possuindo um recuperação de menos de um minuto para desastres entre regiçoes

Fácil de migrar

Migre os bancos de dados mysql ou PostgreSQL para o aurora por meio das ferramentas padrões ou por aplicações do SQL Server herdadas com o Babelshif for AUrora PostgreSQL com minimas de código

Econômico

Alem de melhorar a produtividade o aurora também raduz o custo total da propriedade de um banco de dados totalmente gerenciado incluindo inovações como a tecnologia sem servidor, assim se concentrando nas aplicações
