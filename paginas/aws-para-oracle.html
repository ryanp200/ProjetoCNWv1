Amazon RDS para Oracle

O que é

O Amazon RDS para oracle é um banco de dados comercial totalmente gerenciado que facilita a configuração, a operação, configuração e a escalabilidade de implantações de banco de dados Oracle na nuvem. Com o Amazon RDS é posivel se concentrar na realizalção de inovações e no desenvolvimento enquando ele gerencia as tarefas demoradas de administração de banco de dados, como provisionamento de hardware, aplicação de patches, backup e recuperação, detecção de falhas e reparo.

O RDS para oracle pode ser executado com dois modelos de lincenciamento, sendo elas o modelo "Licença inclusa" onde não é necessário ter adquirido separadamente licenças da Oracle e o "Traga a sua própria licença" (BYOL - Bring Your Own License), onde é possível usar suas licenças existentes do Oracle Database para implantar bancos de dados no Amazon RDS.

Beneficios

Gerenciamento total: O RDS é gerenciado pelo Amazon Relational Database Service o que faz que não seja necessário se preocupar com tarefas de gerenciamento de banco de dados, como provisionamento de hardware, aplicação de patches de software, instalação, configuração e backup

Disponibilidade e confiabilidade O Amazon RDS para Oracle facilita o uso da replicação para aumentar a disponibilidade e a confiabilidade de workloads de produção. Com a opção de implantação Multi-AZ é possivel executar workloads de missão crítica com alta disponibilidade e failover automatizado.

Performance rápida e previsível As instâncias do Amazon RDS suportam até 64 TB de armazenamento integrado em volumes SSD de alta performance do Amazon Elastic Block Store (EBS).

Migração fácil O Amazon RDS for Oracle oferece os benefícios completos de uma solução de serviço gerenciado podendo usar a abordagem "lift-and-shift" para assim migara o banco de dados Oracle para o Amazon RDS for Oracle
